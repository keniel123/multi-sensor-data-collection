Demonstration Videos
====================

Place activity demonstration videos in this folder.

Naming convention:
  <activity_name>.mp4

The activity name must exactly match the "name" column in the
Activities sheet of activities.xlsx.

Examples:
  Activity_Hello.mp4
  Activity_Goodbye.mp4
  Activity_Thank_You.mp4

When an activity is selected in the GUI the first frame of the
matching video is shown as a preview. If no video is found a
placeholder is displayed instead.
